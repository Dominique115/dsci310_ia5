pyospackage is a project that contains functions for adding and multiplying 2 numbers.

## Get started

You can install this package into your preferred Python environment using pip:

```bash
$ pip install pyospackage_dominique115
```

To use pyospackage in your code:

```python
>>> from pyospackage import example
>>> example.add_numbers(2,3)
5
>>> example.mult_numbers(2,3)
6
```
